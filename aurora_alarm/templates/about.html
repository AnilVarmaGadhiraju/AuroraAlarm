About AuroraAlarm Project

AuroraAlarm is a Computer Science student project. This website's purpose it to offer free real-time alarms and photo sharing for Auroras. We tried to focus on ergonomics, usability and simplicity.

We are two Erasmus student spending six months in Luleå. We decided to create this website because we were really excited about Auroras and we couldn't find any good way to be notified for Auroras.


How AuroraAlarm Works

After logging in using Facebook or Google, the user can upload pictures of Auroras and suscribe to alarms, in order to be notified when he's likely to see an Aurora. We distinguish two types of Alarms : Daily alarms and real-time alarms.

Daily alarms are based on the forecast from the Geophysical Institute of University of Alaska Fairbanks. Our website get the daily forecast everyday. Users can set a treshold and receive an email everytime the forecast reaches that treshold.

Real-time alarms rely on the user community. Whenever he's seeing an Aurora, any user can send a notification to every nearby users who activated this option. This feature is based on the different users' current location.



At any time, the user can update his current location and modify his notification settings.


Technology Stack and Architecture

Our architecture relies on a Django server, based on the MVC software pattern. The model is served by a PostgreSQL database. We use Gunicorn as WSGI HTTP Server. Our application is deployed on Heroku, a PaaS adapted to our project.

We use Celery for asynchronous tasks. With Celery, we can schedule regular tasks like parsing the daily forecast. It is also needed for demanding unexpected tasks, for example to send real-time alarms.

The client will access our app with his web browser. He can access our application via HTML requests. Note that interactions with the model will occur through the REST API.


We used the following technologies :

Django
- Photologue for the gallery
- DjangoREST Framework for the REST API
- Celery for asynchronous tasks
Twitter Bootstrap for UI/UX
Facebook/Google API for authentification
Google Maps API
